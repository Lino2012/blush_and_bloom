# 🌸 blush & bloom

A serene digital flower gallery where you can curate your own garden. Built with vanilla HTML, CSS, and JavaScript, featuring smooth 3D card flips, realistic falling petals, and permanent local storage for your personal flower collection.

🔗 **Deployment**: [blush-and-bloom-phi.vercel.app](https://blush-and-bloom-phi.vercel.app)

## ✨ Features

- **Dynamic Flower Gallery** – Browse a curated collection of default flowers (peony, rose, cherry blossom, etc.) with beautiful images and descriptions.
- **3D Flip Cards** – Hover over any flower card to see it flip and reveal a description on the back.
- **Realistic Falling Petals** – Soft pink petals drift down the screen with varied shapes, speeds, and gentle swaying for a calming, immersive effect.
- **Permanent Storage** – Add your own flowers using the form at the bottom. Your additions are saved in `localStorage` and will remain forever (unless you remove them).
- **Admin-Only Deletion** – Only the flowers you add can be deleted; the default collection is protected.
- **Contact Form** – Send a message directly to the admin (demo implementation with EmailJS ready).
- **URL Validation** – When adding a flower, the site checks if the image URL ends with a valid extension (.jpg, .png, etc.) and shows a helpful alert if not.
- **Soft Pink Aesthetic** – A dreamy, cohesive color palette with blurred glass effects and smooth animations throughout.

## 🛠️ Built With

- HTML5
- CSS3 (with CSS variables, animations, backdrop-filter, and 3D transforms)
- Vanilla JavaScript (ES6+)
- [EmailJS](https://www.emailjs.com/) for contact form handling (configured but requires your own keys)
